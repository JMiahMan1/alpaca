💬 Ollama Chat Completions

                                    POST /api/chat
                                

                                

                                    Ollama compatibility endpoint served on Port 11434. Sends chat messages to `llama-server`. Intercepts response stream chunks for live telemetry tracking, filters secrets, and triggers self-healing loops if loading triggers connection loss.
                                

                                
                                
Payload Example

                                
{
  "model": "qwen2.5-coder:7b",
  "messages": [
    { "role": "user", "content": "Write a python quicksort." }
  ],
  "stream": false,
  "options": {
    "temperature": 0.2
  }
}

                                
Curl Command

                                
curl http://localhost:11434/api/chat -d '{"model":"qwen2.5-coder:7b","messages":[{"role":"user","content":"Hi"}],"stream":false}'

                            


                            
                            

                                

                                    
📝 Ollama Text completions

                                    POST /api/generate
                                

                                

                                    Ollama compatibility text generation endpoint served on Port 11434. Includes auto-recovery retry blocks and local sensitive data filters.
                                

                                
                                
Payload Example

                                
{
  "model": "qwen2.5-coder:7b",
  "prompt": "def quicksort(arr):",
  "stream": false
}

                                
Curl Command

                                
curl http://localhost:11434/api/generate -d '{"model":"qwen2.5-coder:7b","prompt":"def f():","stream":false}'

                            


                            
                            

                                

                                    
🏷️ List Local Models

                                    GET /api/tags
                                

                                

                                    Ollama compatibility endpoint on Port 11434. Returns lists of all model configurations registered locally in the manifest store.
                                

                                
Curl Command

                                
curl http://localhost:11434/api/tags

                            


                            
                            

                                

                                    
🏃 List Loaded Models

                                    GET /api/ps
                                

                                

                                    Ollama compatibility endpoint on Port 11434. Returns lists of loaded active models, memory allocation sizes, keep-alive durations, and remaining TTLs.
                                

                                
Curl Command

                                
curl http://localhost:11434/api/ps

                            


                            
                            

                                

                                    
📄 Show Model Information

                                    POST /api/show
                                

                                

                                    Ollama compatibility endpoint on Port 11434. Retrieves metadata details, model parameters, license context, and template format strings for a specific local model.
                                

                                
Curl Command

                                
curl http://localhost:11434/api/show -d '{"name":"qwen2.5-coder:7b"}'

                            


                            
                            

                                

                                    
🔍 Version API

                                    GET /api/version
                                

                                

                                    Ollama compatibility endpoint on Port 11434. Returns standard proxy API compatibility markers.
                                

                                
Curl Command

                                
curl http://localhost:11434/api/version

                            


                            
                            

                                

                                    
🤖 OpenAI Chat completions

                                    POST /v1/chat/completions
                                

                                

                                    OpenAI compatibility chat completion endpoint served on Port 11434. Automatically filters API keys/passwords, registers reasoning metrics, parses thinking blocks (`reasoning_content`), and leverages self-healing recovery loops.
                                

                                
Payload Example

                                
{
  "model": "deepseek-coder:6.7b",
  "messages": [
    { "role": "user", "content": "Explain binary search." }
  ],
  "stream": false
}

                                
Curl Command

                                
curl http://localhost:11434/v1/chat/completions -d '{"model":"deepseek-coder:6.7b","messages":[{"role":"user","content":"Hi"}],"stream":false}'

                            


                            
                            

                                

                                    
📁 OpenAI Models List

                                    GET /v1/models
                                

                                

                                    OpenAI compatibility endpoint on Port 11434. Returns lists of all available router models formatted in standard OpenAI structures.
                                

                                
Curl Command

                                
curl http://localhost:11434/v1/models

                            


                            
                            

                                

                                    
📊 OpenAI Embeddings

                                    POST /v1/embeddings
                                

                                

                                    OpenAI compatibility endpoint on Port 11434. Accepts texts and generates embeddings vectors by calling embed logic of the active backend model.
                                

                                
Payload Example

                                
{
  "model": "nomic-embed-text",
  "input": "text to embed"
}

                                
Curl Command

                                
curl http://localhost:11434/v1/embeddings -d '{"model":"nomic-embed-text","input":"text"}'

                            


                            
                            

                                

                                    
🔍 Fetch Requests History

                                    GET /api/requests
                                

                                

                                    Exposed on Dashboard Port 5000 (proxies to Port 11434 `/admin/requests`). Returns detailed states of all currently active requests and the 10 most recently completed requests. Credentials are automatically redacted.
                                

                                
Curl Command

                                
curl http://localhost:5000/api/requests

                            


                            
                            

                                

                                    
🗑️ Clear Telemetry History

                                    POST /api/requests/clear
                                

                                

                                    Exposed on Dashboard Port 5000 (proxies to Port 11434 `/admin/requests/clear`). Clears the finished completed requests history logs. Active running requests are unaffected.
                                

                                
Curl Command

                                
curl -X POST http://localhost:5000/api/requests/clear

                            


                            
                            

                                

                                    
🖥️ Fetch Hardware Telemetry

                                    GET /admin/system
                                

                                

                                    Served on Port 11434. Queries real-time hardware status metrics from the host machine, including CPU load, RAM usage, and GPU VRAM indicators.
                                

                                
Curl Command

                                
curl http://localhost:11434/admin/system

                            


                            
                            

                                

                                    
⚙️ Fetch Loaded Models Context

                                    GET /admin/runtime
                                

                                

                                    Served on Port 11434. Fetches runtime states of all loaded models inside the router, their KV cache context size utilization, memory allocation details, and TTL expiration timestamps.
                                

                                
Curl Command

                                
curl http://localhost:11434/admin/runtime

                            


                            
                            

                                

                                    
🎛️ Fetch Llama Server Slots

                                    GET /admin/slots
                                

                                

                                    Served on Port 11434. Directly queries current speculative/execution slots from the underlying `llama-server` process, showing token generation speeds and active slots context length metrics.
                                

                                
Curl Command

                                
curl http://localhost:11434/admin/slots

                            


                            
                            

                                

                                    
📊 Fetch Metrics and Throughput

                                    GET /admin/metrics
                                

                                

                                    Served on Port 11434. Returns general proxy benchmarks, request latency statistics, prompt token metrics, and completion token metrics.
                                

                                
Curl Command

                                
curl http://localhost:11434/admin/metrics

                            


                            
                            

                                

                                    
📜 Fetch Proxy Logs

                                    GET /api/logs
                                

                                

                                    Served on Port 11434. Retrieves raw logs output from the alpaca-proxy daemon stack. Supports a `limit` query parameter.
                                

                                
Curl Command

                                
curl http://localhost:11434/api/logs?limit=50

                            


                            
                            

                                

                                    
📁 Fetch Model Profiles

                                    GET /api/profiles
                                

                                

                                    Served on Dashboard Port 5000. Reads and parses all defined configuration sections inside `models.ini`, including context cap sizes, thread count, VRAM budgeting layers, and speculative parameters.
                                

                                
Curl Command

                                
curl http://localhost:5000/api/profiles

                            


                            
                            

                                

                                    
💾 Save/Update Model Profile

                                    POST /api/profiles/save
                                

                                

                                    Served on Dashboard Port 5000. Saves or modifies individual parameter keys for a designated model section inside `models.ini`.
                                

                                
Payload Example

                                
{
  "section": "qwen2.5-coder:7b",
  "settings": {
    "n_gpu_layers": "32",
    "flash_attn": "true"
  }
}

                                
Curl Command

                                
curl http://localhost:5000/api/profiles/save -d '{"section":"qwen2.5-coder:7b","settings":{"n_gpu_layers":"32"}}'

                            


                            
                            

                                

                                    
❌ Delete Model Profile

                                    POST /api/profiles/delete
                                

                                

                                    Served on Dashboard Port 5000. Deletes a custom profile section from `models.ini`. The default general `[*]` configuration section cannot be deleted.
                                

                                
Payload Example

                                
{
  "section": "qwen2.5-coder:7b"
}

                                
Curl Command

                                
curl http://localhost:5000/api/profiles/delete -d '{"section":"qwen2.5-coder:7b"}'

                            


                            
                            

                                

                                    
🔄 Restart Backend Services

                                    POST /api/proxy/restart
                                

                                

                                    Served on Dashboard Port 5000. Triggers background container restart sequences for both `llama-server` and `alpaca-proxy` containers.
                                

                                
Curl Command

                                
curl -X POST http://localhost:5000/api/proxy/restart

                            


                            
                            

                                

                                    
📊 Fetch Benchmark Results

                                    GET /api/results
                                

                                

                                    Served on Dashboard Port 5000. Returns detailed lists of all saved standard benchmarks and SharedLLM verification result runs from disk.
                                

                                
Curl Command

                                
curl http://localhost:5000/api/results
